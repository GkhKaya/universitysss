# universitysss

Üniversite ortamında soru–cevap ve bilgi paylaşımı için geliştirilen web uygulaması. Ön yüz **React**, **TypeScript** ve **Vite** ile yazılmıştır; veri ve kimlik için **Firebase** (Auth, Firestore) kullanılır. Kod yapısı feature-based düzenlenmiştir; Firestore koleksiyonları için tip tanımları `src/shared/types` altında tutulur.
